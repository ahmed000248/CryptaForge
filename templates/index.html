What is Classical Cryptography?

Before the digital age of computer algorithms (like AES or RSA), secrets were protected using manual, paper-and-pencil methods called classical ciphers. These systems are broadly categorized into two types:

Substitution Ciphers

These ciphers replace letters of the plaintext with other letters, numbers, or symbols. Caesar, Playfair, Hill, and our Custom Alphabet builder fall under this category.

Transposition Ciphers

Instead of replacing letters, transposition ciphers rearrange the order of letters according to a systematic grid, key, or path. Rail Fence and Columnar Transposition fall under this category.

Implemented Algorithms

A complete suite of classical algorithms with full decryption support.

Very Weak
Caesar Cipher

Shift letters by a numeric key index (0-25). Excellent starting point for basic shift mechanics.

Weak
Playfair Cipher

Substitute character digraphs using a 5x5 key matrix. Protects against basic frequency analysis.

Moderate
Hill Cipher

Encrypt blocks of text using matrix multiplication (2x2 key). Explains linear algebra applications.

Moderate
Columnar Transposition

Permute character layout using key-based sorting of columns. Demonstrates spatial shuffling.

Weak
Rail Fence Cipher

Write text diagonally across multiple 'rails' and read it horizontally. Visualizes pattern-based sorting.

Custom Cipher

Construct your own alphabet permutation. Analyze substitution weaknesses through customization.
